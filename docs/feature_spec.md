# ChartSkin — 프로젝트 피처 스펙

> 차트 위에 스킨(테마)을 입혀서 나만의 트레이딩 뷰를 만드는 웹앱

---

## 제품 한 줄 정의

> 주식·코인 차트를 불러오고, 마켓플레이스에서 테마를 골라 입히는 웹앱.
> 배경·캐릭터·말풍선 등 감성적인 스킨으로 차트를 꾸미고, AI로 나만의 스킨을 만든다.

---

## 전체 화면 구조

```
┌────────────────────────────────────────────┬──────────────────┐
│                                            │                  │
│   헤더: 종목 검색 / 마켓 선택               │   사이드바        │
│                                            │   마켓플레이스    │
├────────────────────────────────────────────│                  │
│                                            │  [테마 카드들]   │
│              차트 영역                      │  - 배경 스킨     │
│                                            │  - 지표 스킨     │
│   ┌─────────────────────────────────────┐  │  - 세트 테마     │
│   │  배경 스킨 레이어 (고정)              │  │                  │
│   │  + 차트 캔들/라인                    │  │  [적용 중인 스킨] │
│   │  + 지표 스킨 레이어 (좌표 추적)      │  │  - 수정 / 해제   │
│   └─────────────────────────────────────┘  │                  │
│                                            │                  │
│   거래량 차트                               │                  │
└────────────────────────────────────────────┴──────────────────┘
```

## 스킨 타입 정의

### 1. 배경 스킨 (Background Skin)
- 차트 뒤에 깔리는 이미지/일러스트
- 차트 스크롤·줌에 **고정** (움직이지 않음)
- 투명도 조절 가능 (차트 가독성 확보)
- 예시: 밤하늘, 장원영 사진, 귀여운 패턴

### 2. 지표 스킨 (Indicator Skin)
- 특정 **가격 포인트**에 캐릭터/아이콘을 붙이는 스킨
- 차트 스크롤·줌 시 **좌표 추적** (가격축 연동)
- 바인딩 포인트 종류:
  - 기간 내 최고점 / 최저점
  - 매수 / 매도 지점 (사용자 직접 지정)
  - 이동평균선 골든크로스 / 데드크로스
  - 사용자 정의 가격선
- 캐릭터 상태: 상승 시 happy, 하락 시 sad, 횡보 시 neutral (표정 variant)
- 말풍선 텍스트 커스터마이징 가능

### 3. 세트 테마 (Theme Set)
- 배경 스킨 + 지표 스킨 + 차트 색상 + 말풍선 스타일을 묶은 패키지
- 예시: "밤하늘 테마", "귀엽고 달달한 테마", "사이버펑크 테마"

## 피처 목록

### Phase 1 — MVP

#### 1-1. 차트 뷰어
- [ ] 종목/코인 검색 (심볼 검색)
- [ ] 마켓 선택: 암호화폐 / 국내 주식 / 미국 주식
- [ ] TradingView Lightweight Charts 임베드
- [ ] 기본 캔들차트 + 거래량 차트
- [ ] 이동평균선 (5, 20, 60, 120일)
- [ ] 기간 선택 (일/주/월/년, 1분/5분/15분/1시간)
- [ ] 줌, 스크롤, 십자선 커서

#### 1-2. 사이드바 마켓플레이스
- [ ] 테마 카드 목록 (썸네일 + 이름 + 태그)
- [ ] 카테고리 필터: 배경 스킨 / 지표 스킨 / 세트 테마
- [ ] 태그 필터: 귀여운, 어두운, 미니멀, 사이버펑크 등
- [ ] 테마 클릭 → 차트에 즉시 프리뷰 적용
- [ ] 적용 / 해제 버튼
- [ ] 현재 적용 중인 스킨 표시

#### 1-3. 배경 스킨 렌더링
- [ ] 선택한 이미지를 차트 뒤 레이어에 렌더링
- [ ] 투명도 슬라이더
- [ ] 이미지 fit 모드: cover / contain / tile

#### 1-4. 지표 스킨 렌더링
- [ ] 가격 좌표 → 화면 픽셀 좌표 변환 (실시간)
- [ ] 최고점 / 최저점 자동 감지 및 캐릭터 표시
- [ ] 말풍선 텍스트 기본 제공
- [ ] 스크롤·줌 시 위치 자동 재계산

#### 1-5. 차트 내보내기
- [ ] 현재 차트 + 스킨 PNG 저장
- [ ] 저장 시 해상도 선택 (1x / 2x)

---

### Phase 2 — 스킨 시스템 고도화

#### 2-1. 스킨 에디터
- [ ] 드래그앤드롭으로 캐릭터 위치 조정
- [ ] 바인딩 포인트 선택 UI (어떤 지표에 붙일지)
- [ ] 말풍선 텍스트 직접 입력
- [ ] 캐릭터 크기 조절
- [ ] 감정 상태별 이미지 업로드 (happy / sad / neutral)
- [ ] 내 스킨으로 저장

#### 2-2. 감정 반응 자동화
- [ ] 기준 기간 대비 등락률로 감정 상태 자동 결정
- [ ] 실시간 업데이트 (틱 데이터 수신 시)

#### 2-3. 차트 색상 테마
- [ ] 캔들 색상 커스터마이징 (상승/하락)
- [ ] 배경색, 그리드색, 텍스트색
- [ ] 이동평균선 색상/굵기

#### 2-4. 애니메이션 스킨
- [ ] GIF / Lottie 형식 지원
- [ ] 가격 변동 시 애니메이션 트리거

#### 2-5. 비디오 클립 내보내기
- [ ] 실시간 차트 움직임 + 스킨을 MP4로 녹화

---

### Phase 3 — AI 스킨 생성

#### 3-1. 스타일 기반 생성
- [ ] 텍스트 프롬프트로 배경 스킨 생성
  - 예: "밤하늘에 별이 가득한 몽환적인 분위기"
- [ ] 스타일 태그 선택형 UI (프롬프트 보조)
- [ ] 생성 결과 여러 장 중 선택

#### 3-2. 캐릭터 스킨 생성
- [ ] 스타일 선택 → 캐릭터 생성
- [ ] 감정 variant 자동 생성 (happy / sad / neutral 3종)
- [ ] 배경 자동 제거 (rembg 또는 API)

#### 3-3. 커스텀 이미지 업로드 → 스킨화
- [ ] 사용자가 이미지 직접 업로드
- [ ] 배경 제거 + 스킨 포맷으로 변환
- [ ] 캐릭터로 등록 후 에디터에서 사용

## 확정된 결정 사항

| 항목 | 결정 |
|------|------|
| 스킨 저장 위치 | 로컬 스토리지 (서버 불필요) |
| KRX 데이터 | 전일 종가 데이터로 충분 (실시간 불필요) |
| 마켓플레이스 초기 콘텐츠 | 직접 제작 프리셋 스킨 10개로 시작 |
| 사용자 계정 | 로그인 없음 |
| 모바일 대응 | 데스크탑 전용 (반응형 불필요) |

→ **백엔드/DB 불필요.** 프론트엔드 단독으로 완성 가능한 구조.

## 데이터 소스 계획

| 마켓 | API | 비고 |
|------|-----|------|
| 암호화폐 | Binance REST API | 무료, 일봉 데이터 |
| 미국 주식 | Polygon.io | 무료 티어 (전일 데이터) |
| 국내 주식 (KRX) | 네이버 금융 / KRX 공공데이터 포털 | 전일 데이터, 계좌 불필요 |

## 기술 스택

| 레이어 | 기술 |
|--------|------|
| 프론트엔드 | React + TypeScript |
| 차트 | TradingView Lightweight Charts v4 |
| 스킨 렌더링 | Canvas 2D (차트 위 오버레이) |
| 스타일링 | Tailwind CSS |
| 상태관리 | Zustand |
| 스킨 저장 | localStorage (JSON 직렬화) |
| AI (Phase 3) | Replicate API (Stable Diffusion) 또는 OpenAI DALL-E |
| 백엔드 | **없음** — 순수 프론트엔드 앱 |
